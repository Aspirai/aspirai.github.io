CSS文本控制

实现文本竖向排列的方法有多种，主要取决于具体需求。以下是几种常见的方法：

1. 使用 CSS writing-mode 属性

writing-mode 可以将文本流从水平改为垂直排列。通常用于需要整个段落竖向显示的情况。

  • vertical-rl：文本从右向左垂直排列。
  • vertical-lr：文本从左向右垂直排列。
  • text-orientation: upright：将字符保持正立。

2. 使用 Flexbox 并旋转每个字母

适合单词或短句的竖向排列。通过 flex-direction: column 将每个字母或字符设置成一列。

3. 使用 transform 属性旋转整个容器

可以将容器旋转 90 度来实现文本垂直显示，适合块级文本段落。

这种方法会整体旋转文本，需要在排版时注意旋转方向和定位。

4. 使用 CSS Grid 设置单列布局

将每个字母或单词放入单独的 <span> 元素中，并通过 CSS Grid 进行单列布局。

选择合适的方式取决于具体的文本内容和设计需求。

可以通过 CSS 控制文本不超过父元素，以下是几种常见的方法：

1. 使用 overflow: hidden 和 text-overflow: ellipsis

  • 适用于单行文本，超出部分以省略号显示。

2. 多行文本省略

  • 可以使用 display: -webkit-box 配合 -webkit-line-clamp 实现多行文本省略。适用于多行文本，超出行数时以省略号显示。

3. 限制 max-height 和 overflow 实现裁剪

  • 适用于让文本内容不超出指定的高度（比如让文本超出父元素时隐藏）。

4. 使用 flex 或 grid 配合 overflow

  • 可以在父元素上应用 display: flex 或 display: grid，然后配合 overflow，让文本在父元素中自动换行且不超出。

这些方法可以根据需求选择，适用于单行或多行文本，确保文本在父元素边界内显示。
