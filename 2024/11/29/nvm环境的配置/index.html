    原因: 权限不够;

    解决: 使用管理员权限进行nvm install xxxx

    原因: node版本过高

    解决: 降低node

依赖安装报错

  1. node与npm版本不兼容

  2. 网络问题

  3. 权限问题

    需要管理员权限,在全局安装时;

  4. 锁文件冲突

    package-lock.json 或 yarn.lock 可能导致依赖版本冲突或错误

  5. npm缓存问题

  6. 再进行node与npm版本适配后,重新安装了umi
