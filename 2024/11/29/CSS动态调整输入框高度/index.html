CSS动态调整输入框高度

在 Taro 1.x 中，由于不能使用 min-width 来实现输入框随内容自动变宽，可以通过以下方法来实现：

方法一：使用 width: auto 和 placeholder 补充

确保输入框的宽度动态调整，并使用 placeholder 提示用户。

方法二：根据内容动态计算宽度（最佳方案）

通过绑定输入框的值到一个隐藏的 Text 标签，用其内容宽度动态设置输入框宽度。

核心思路：

  1. 使用隐藏 Text 标签测量输入的内容宽度。
  2. 使用 Input 组件动态调整宽度。

JS 逻辑：

方法三：使用 flex 布局和 contenteditable

如果允许，可以使用 contenteditable 元素代替输入框，其天然支持内容自适应宽度。

推荐方案

方法二 是最推荐的实现方式：

  • 它兼容性强且动态。
  • 样式保持灵活性。
  • 适合 Taro 框架的 UI 开发。
